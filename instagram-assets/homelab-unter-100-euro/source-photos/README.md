# Source Photos — Homelab unter 100€

Lege hier deine eigenen echten Fotos ab.

**Erwartetes Format:** JPEG oder PNG  
**Mindestauflösung:** 1920 × 1920 Pixel  
**Dateinamen:** beschreibend, z. B. `thin-client-vorne-1.jpg`

## Ablauf nach Upload

1. Hermes sichtet die Fotos und wählt die besten aus
2. Erstellt 4:5 (1080×1350) und 9:16 (1080×1920) Crops
3. Baut Overlays (Text, Icons, Grafiken) nach Caption-Vorlage
4. Erstellt 3 Layout-Varianten pro Instagram-Asset
5. Erzeugt finale Review-Galerie aus echten Fotos
6. Ersetzt KI-Konzeptbilder durch echte Fotos
