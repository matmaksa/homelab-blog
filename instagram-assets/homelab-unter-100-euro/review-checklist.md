## 1. Blog-URL-Konsistenz

- [ ] Finale Artikel-URL korrekt: `https://matmaksa.github.io/homelab-blog/posts/homelab-unter-100-euro-was-du-brauchst/`
- [ ] Keine Preview-URL (`/preview/...`) in Captions, CTAs oder Metadaten
- [ ] Alle Link-in-Bio-Verweise zeigen auf die finale Blog-URL
- [ ] Keine toten oder veralteten Links

## 2. Hardware-Korrektheit

- [ ] Kein S920 als Hauptbeispiel
- [ ] S740/S7010 nur als Homelab-/Proxmox-/Selfhosting-Beispiele (keine KI-Empfehlung)
- [ ] Keine KI-/Ollama-Kaufempfehlung aus den Thin Clients
- [ ] Keine übertriebenen Leistungsversprechen

## 3. Affiliate & Disclosure

- [ ] Keine Affiliate-Links in Captions oder CTAs
- [ ] Keine Affiliate-Offenlegung nötig (da keine Affiliate-Inhalte)
- [ ] Keine Produktempfehlungen mit Kaufdruck

## 4. Caption-Qualität

- [ ] Caption verständlich und direkt (nicht schwammig)
- [ ] Hook in Zeile 1 stark genug (Neugier, Problem, Überraschung)
- [ ] Caption ≤ 200 Zeichen (gemessen)
- [ ] Keine künstliche KI-Sprache ("Tauche ein in...", "In der heutigen digitalen Welt...")
- [ ] Anfängerfreundlich (kein übermäßiges Fachchinesisch)
- [ ] Keine Rechtschreib-/Grammatikfehler
- [ ] Emojis sinnvoll und nicht überladen (max 2-3)

## 5. Hashtags

- [ ] Maximal 8 Hashtags
- [ ] Hashtags relevant zum Thema (Homelab, Einsteiger, Technik)
- [ ] Keine irrelevanten Massen-Hashtags (#love #instagood etc.)
- [ ] Hashtags lesbar (keine zusammengewürfelten Buchstaben)

## 6. Visual-Qualität

- [ ] Bild/Cover zum Derivat-Thema passend
- [ ] Format korrekt:
  - [ ] Reel = 9:16
  - [ ] Carousel = 4:5 (alle Slides einheitlich)
  - [ ] Story = 9:16
- [ ] Text im Bild kurz und lesbar (max 15 Wörter pro Slide)
- [ ] Text im Bild kontrastreich (lesbar auf dem Hintergrund)
- [ ] Blog-Cover wurde nicht als Instagram-Bild verwendet (16:9 ≠ IG-Asset)
- [ ] Keine generischen KI-Serverräume oder Cyberpunk-Ästhetik

## 7. CTA-Prüfung

- [ ] Jeder Post hat einen klaren CTA
- [ ] CTA passt zum Ziel des Posts:
  - [ ] Save-Post → "Speichern für später"
  - [ ] Kommentar-Post → "Schreib's in die Kommentare"
  - [ ] Traffic-Post → "Link in Bio"
- [ ] CTA nicht aufdringlich (kein Hard-Sell)

## 8. Veröffentlichungs-Kontrolle

- [ ] Kein direkter Instagram-Post ohne vorherige Freigabe
- [ ] Posting-Reihenfolge eingehalten (keine Überschneidungen)
- [ ] Bio-Link aktualisiert, falls nötig
- [ ] Story-Hinweise vorbereitet (falls Stories parallel laufen)
